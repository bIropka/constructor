На данный момент готовы 4 вида сетки:

product-grid-v-lg
product-grid-h-lg
product-line-h-lg
product-grid-v-sm

Название класса состоит из 4-х частей:
1) product - название родительского блока
2) grid/line - расположение товаров (таблицей / в линию)
3) h/v - расположение элементов в блоке товара (горизонтально / вертикально)
4) lg/sm - наполнение блока товара (полная / сокращенная версии)

Все элементы независимы, то есть могут добавляться в любом порядке не разарушая верстку.

.wrapper - контейнер верхнего уровня (родительский для остального контента)
могут добавляться классы content-width-small, content-width-medium которые изменяют ширину контентной части;

.wrapper-inner - определяет ширину контентной части страницы в зависимости от класса контейнера верхнего уровня;

сайдбары следует размещать в таком порядке: sidebar-before, sidebar-after, затем content-block
